Tweeter Mining

In this blog, I want to explain the concept of tweeter mining, getting feature attributes desired using tweepy API and doing exploratory analysis on the dataset obtained.

We will also see some of the ways to create visualizations.

This project was funded by 10 Academy as part of my training.

Introduction

Social media campaigns are mostly driven by influencial individuals in the network and therefore identifying them becomes such a big win. This project was undertaken to find out the most influential twitter users, which hashtags are used, and what parameters are used to identify influential rate of a twitter user. Parameters used for the task were:

  • Popularity score: measured by the number of retweets and likes users get
  • Reach score: measured by the size of audience a single user has (comprise of followers and the people they follow)
  • Relevance score: measured by the number of mentions a single user has.

Methodology

Data Extraction

To investigate the goal mentioned above, I obtained data by scraping Africa Influential Tweeter Users and Africa leaders response to Corona virus websites. I extracted twitter handles from the data obtained from these websites which formed two lists for twitter influential users and government officials respectively.

Data Processing

Since the data was not all clean, I started by doing data wrangling to ensure I had the data in good shape. This included dropping rows with missing values since other methods for handling missing values were not fit for the rows. I also did feature engineering so as to obtain the result I needed.

Data Analysis

I did visualization on the clean to data to help get insight of the trend in the data. The main focus was on the three ranking scores of measuring influence; popular score, Relevance score and Reach score.

Conclusion and Recommendation

Retweet influence, mentions influence and indegree influence have big a role to play in determining the level of influence a user has on the society and its environment. And this is a strength that can be tapped into and utilized by businesses to push products and services through campaigns by partnering with the most influential users through marketing and advertising.
